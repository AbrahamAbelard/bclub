# Security Guidelines

Security is an important part of every modern website.

## General Recommendations

Visitors should always:

- Verify the website address before signing in.
- Keep account credentials private.
- Use strong passwords.
- Enable additional protection whenever available.

## Safe Browsing

Before entering account information, make sure:

- The website address is correct.
- HTTPS is enabled.
- The connection is secure.

## Reporting Issues

If you discover a security issue related to the website, please contact the support team through the official website.
